High Resoltion Zelda Maps

All images have a resolution of 24064 x 18944. Credit to zeldamods.org for the source. These were generated with a program I made that scraped the several thousand max-zoom images from each game's interactive map on the site. I then stitched them together into a single large image.

Pros and Cons of PNG and AVIF image versions

  • PNG:
    • Pros:
      • Uses less ram while loading (Approx. the size of the image)
      • Slightly less compression artifacts
      • More software support
    • Cons:
      • Takes up substantially more storage space

  • AVIF:
    • Pros:
      • Takes up substantially less storage space
    • Cons:
      • Uses more ram while loading (Approx. 6 GB)
      • Slightly more compression artifacts
      • Less software support
